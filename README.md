## BRCAness view
Displays a patient's BRCAness score on a mirrored histogram. Contextualizes the patient's score among those with the same cancer (upper histogram) and those with ovarian cancer (lower histogram).

## Overview
This project was written in javascript and bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
It uses react, victory, react-bootstrap, and d3-array.


## To view in browser
```
git clone https://github.com/hallegakoen/brcaness-view.git
```
```
cd brcaness-view
```
```
npm install
```
```
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
